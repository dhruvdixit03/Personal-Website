PROJECTS

Pokemon TCG Game

I created this game using Python and external libraries such as pygame. The game uses an advanced Monte Carlo Algorithm in the Hard Mode so that the opponent effectively chooses the best possible move to use. Object-oriented-programming was also used to create all pokemon classes and give them attributes such as health, attack, type, etc. OOP was also used to create methods of attack.
